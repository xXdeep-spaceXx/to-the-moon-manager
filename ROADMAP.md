# Roadmap

Last updated: 2026-02-18

## Phase 1: Core Prototype (Now)
- Task CRUD with metadata (category, difficulty, priority)
- XP/leveling + streaks
- Local persistence
- Basic insights and recommendations

## Phase 2: Sync + Accounts
- Auth (email/password) and session tokens
- Server-backed state sync
- Conflict handling and last-write-wins
- Multi-device support

## Phase 3: Game Layer Expansion
- Ship upgrades and unlockables
- Skill trees and perks
- Daily/weekly challenges
- Streak rewards and penalties

## Phase 4: Adaptive Learning
- Personalized XP tuning
- Habit detection and recommendations
- Task difficulty calibration
- Goal-based mission generation

## Phase 5: Mobile + App Store
- React Native or Flutter app
- Offline-first architecture
- Secure storage + device keychain
- Analytics with privacy controls
- App Store compliance and review prep

## Risks & Dependencies
- User data privacy and encryption
- Billing + subscriptions if monetized
- Sync conflicts across devices
- Model-based recommendations (cost + safety)
